### Requisitos para baixar o projeto
- JAVA
- JDK
- MySQL
- Netbeans 8
- Git
   * Se ainda não tiver baixado, recomendo procurar um vídeo no YT para configurar

### Sequência para baixar o projeto no NetBeans
Abra o NetBeans 8 e acesse:
  EN **Team → Remote → Clone** 
  PT **Equipe → Remoto → Clonar**
